# Space README Template Reference

Use the full Space README template from the account-surface Hugging Face template set until `foundation-spaces` is scaffolded.

Required public sections:

- purpose;
- demo scope;
- what the demo does;
- what the demo does not do;
- model and dataset dependencies;
- safety limits;
- public/private boundary;
- GitHub companion link;
- Foundation context;
- status badge placeholder.
