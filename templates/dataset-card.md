# Dataset Card Template Reference

Use the full dataset card template in `foundation-dataset-cards/templates/dataset-card.md`.

Required public sections:

- dataset summary;
- data source;
- collection process;
- allowed uses;
- forbidden uses;
- privacy exclusions;
- PII statement;
- bias and limitations;
- governance review status;
- maintenance plan;
- GitHub and Hugging Face links;
- Foundation context.
