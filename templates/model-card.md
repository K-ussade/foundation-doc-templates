# Model Card Template Reference

Use the full model card template in `foundation-model-cards/templates/model-card.md`.

Required public sections:

- model summary;
- intended use;
- out-of-scope use;
- Foundation alignment;
- training data summary;
- data exclusions;
- privacy boundaries;
- evaluation status;
- limitations;
- bias and risk notes;
- safety notes;
- citation, contact, governance;
- GitHub and Hugging Face links;
- version history.
