# Security Policy

## Reporting Sensitive Issues

Do not open a public issue containing secrets, private records, donor data, student data, school private data, private incident information, private telemetry, infrastructure details, or sealed YOSO-YAi LLC IP.

Use a private maintainer contact path until a formal Foundation security contact is published.

## Public Issue Scope

Public issues may discuss:

- template wording;
- placeholder clarity;
- broken links;
- graph rendering;
- validation checks;
- public boundary improvements.

Public issues must not include:

- credentials, tokens, or API keys;
- real private data;
- private governance drafts;
- private incident records;
- model weights or private evaluations;
- raw datasets or private telemetry;
- security-sensitive NEURONA operational details.

## Supported Scope

This repository is documentation-only. It does not collect data, run systems, host models, publish datasets, or operate demos.
